\subsection{\-Detailed \-Description}
\-A simple class designed so that a circular buffer can be imitated with reasonable speed. 

rotator \-This takes in strings and writes them into the files name name1 name2 ... namen such that if enough elements are written everything will be renamed and namen will be overwritten with new entries.
